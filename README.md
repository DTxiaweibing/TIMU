本应用包含三大核心模块，覆盖养殖生产与技术学习全流程，全程无需网络，数据本地存储。

A feed intake recording tool designed specifically for shrimp farms.
时间输入 不用输冒号，直接输数字自动格式化
Time input: no colon needed, numbers are automatically formatted
登记吃完时间时，输入任意字符自动变成对号，并自动算出吃料用时
When recording finish time: any input automatically becomes a check mark, and feeding duration is calculated automatically
支持“去拌料”标记、异常棚号跳过计算
Supports "Go Mix Feed" tag and skips abnormal pond numbers in calculation
颜色区分状态，一目了然，适合大规模养殖管理
Color-coded status for clear overview, suitable for large‑scale farming management

告别纸笔计算器，全表自动计算，操作简单直观。
Say goodbye to pen and paper calculators; full table auto‑calculation with simple and intuitive operation.
多规格自动计算：发酵料、粉料、0.3料、0.5料、1.0料、加水量
Auto‑calculation for multiple specifications: fermented feed, powder feed, 0.3 feed, 0.5 feed, 1.0 feed, water volume
可自定义加水比例（10%–40%）
Customizable water ratio (10%–40%)
棚数超出预设时，自动 红边框 + 黄背景 醒目提醒

Supports multi‑pond batch calculation, column fixing, hiding and other advanced table operations
