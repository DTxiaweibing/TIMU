# 🦐 养虾助手 (Shrimp Farming Assistant)

## 📱 核心功能

本应用包含三大核心模块，覆盖养殖生产与技术学习全流程，**全程无需网络**，数据本地存储。

### 1. 🧮 养虾拌料计算器
告别纸笔与计算器，一键完成多棚配料计算。
- **多规格支持**：自动计算发酵料、粉料、0.3料、0.5料、1.0料及水分比例。
- **灵活设置**：支持自定义水百分比（10% - 40%），发酵料参与算水/算料自由切换。
- **批量计算**：支持多棚号批量录入，自动汇总称料总量，支持列固定/隐藏等表格高级操作。

### 2. 📝 任意棚数查料表
精准记录每一口棚的吃料情况，掌握虾苗生长动态。
- **时间追踪**：自动记录开始喂料、结束喂料、查料时间，精准计算**吃料用时**。
- **状态标记**：支持“去拌料”快速标记，异常棚号可标记“本行不参与计算”。
- **数据清晰**：通过颜色区分不同状态，支持批量清除与确认，适配大规模养殖管理。

### 3. 📚 养殖刷题宝
利用碎片时间巩固养殖技术，新手入门、老手进阶的必备工具。
- **专业题库**：内置《对虾养殖考题》等专属题库，覆盖水温临界值等核心实操知识点。
- **题库管理**：支持本地导入自定义题库与在线题库下载，多题库自由切换。
- **答题反馈**：实时判断对错并提供详细解析，记录答题进度与正确率。
